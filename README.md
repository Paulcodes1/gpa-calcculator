# 🎓 Academic GPA Computation Tool
**An institutional-grade GPA calculator mimicking the SAFRecords module.**

## 🚀 Features
- **Tabular Data Entry:** Dynamic rows for Course Code, Units, and Grades.
- **Persistent Storage:** Uses `LocalStorage` to save data between sessions.
- **AI Performance Advisor:** Predictive logic to suggest grades needed for a target GPA.
- **Print-Ready:** Specialized CSS media queries for generating clean PDF reports.

## 🛠️ Technologies Used
- **HTML5:** Semantic structure for tabular data.
- **CSS3:** Custom properties (variables) and Responsive Grid Layout.
- **JavaScript (ES6):** DOM Manipulation, Array Methods (`map`, `filter`, `reduce`), and LocalStorage API.
